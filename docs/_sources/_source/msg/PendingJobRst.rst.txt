PendingJob msg
==============

The `PendingJob.msg`_ message type is used to send information about a specific Pending Job, Pending meaning a MobileExecutor is yet to be assigned to this Job.

It contains information such as the Job ID, it's priority, the number of tasks it has, and a keyword signifying the type of Job.
